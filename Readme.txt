Sentiment Classifier for Small Scale Businesses
The Sentiment Classifier for Small Scale Businesses is a software application designed to provide sentiment analysis for product reviews. 
This project aims to help small business owners gain insights into customer sentiments, facilitating better product development and decision-making.

Table of Contents

Introduction
Customer feedback plays a crucial role in the success of any business. Understanding the sentiments expressed by customers in product reviews 
can be vital in improving product offerings and enhancing customer satisfaction. This sentiment classifier software employs machine learning techniques 
to classify product reviews into positive, negative, or neutral sentiments.

Features
User-friendly frontend interface for entering product reviews.
Sentiment analysis powered by advanced machine learning algorithms.
Option to export sentiment analysis results to a CSV file for further analysis and decision-making.
Scalable and cost-effective solution tailored to small scale businesses.

Usage
Navigate to the project directory and run the application:
bash
Copy code
cd sentiment-classifier
python app.py
Access the application via your web browser at http://localhost:5000.
Enter the product reviews in the provided input box and submit for sentiment analysis.
The software will classify the sentiments and display the results on the frontend.
Optionally, choose to export the sentiment analysis results to a CSV file for further analysis.
